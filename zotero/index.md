# Zotero的使用


<!--more-->

Zotero

Zotero是一款开源文献管理软件，功能和EndNote相似，但是其丰富的插件使其具有十分强大的功能。

到官网（[Zotero | Downloads](https://www.zotero.org/download/)）下载对应版本的Zotero和浏览器插件。

按照网上教程安装Zotero插件，参考连接如下：

[Zotero的插件介绍 - 知乎 (zhihu.com)](https://zhuanlan.zhihu.com/p/361584342)

安装quicklook后点击空格就可以预览。

[Zotero网盘同步及文件夹重命名 - 知乎
(zhihu.com)](https://zhuanlan.zhihu.com/p/449527854)

所有文件都存储到了onedrive的文件夹下。

下载坚果云客户端，配置文件夹可以达到同样的效果。
